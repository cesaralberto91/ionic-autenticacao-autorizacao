# Ionic: Autenticação e Autorização
==============================================

Usage
-----
Clone repo
> git clone https://github.com/cesaralberto91/ionic-autenticacao-autorizacao.git

Configure
> cd ionic-autenticacao-autorizacao

> npm install

Start APP
> ionic serve
